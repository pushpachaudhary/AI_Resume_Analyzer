# AI_Resume_Analyzer 🚀
AI-powered Resume Analyzer that extracts skills, evaluates resumes, and provides improvement suggestions using Flask and NLP.

## 📌 Features
- 📄 Upload Resume (PDF)
- 🧠 AI-based Skill Extraction
- 📊 Resume Score Generation
- 💡 Smart Suggestions for Improvement
- 🌐 Interactive Web Interface

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Flask (Python)  
- **AI/NLP:** Python, NLTK  
- **Other Tools:** PyPDF2  

---

## ⚙️ How It Works
1. User uploads a resume (PDF)
2. Text is extracted using PyPDF2
3. AI analyzes skills using NLP techniques
4. System generates:
   - Extracted Skills
   - Resume Score
   - Improvement Suggestions
